# Premium Saree Showcase

A luxury Indian Saree E-Commerce Website built with React, featuring premium UI/UX design, smooth animations, and a complete admin dashboard.

## Tech Stack

- React 18 with Vite
- React Router
- Bootstrap 5
- AOS (Animate On Scroll)
- GSAP (Animation)
- Framer Motion (Page Transitions)
- Lenis (Smooth Scrolling)

## Features

- Premium luxury design
- Smooth animations and transitions
- Product catalog with filters
- Shopping cart and wishlist
- Admin dashboard
- Razorpay payment integration ready
- Responsive design

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
